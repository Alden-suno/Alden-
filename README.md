# Alden Frontend (React + Tailwind)

This is a React + Tailwind starter frontend for **Alden** â€” a business directory and subscription platform.

## Quick start (local)
1. Install dependencies:
```
npm install
```
2. Run dev:
```
npm run dev
```
3. Build:
```
npm run build
```

## Environment variables
Create a `.env` file in the project root if needed:
```
VITE_API_URL=https://alden-backend.onrender.com
REACT_APP_FLW_PUBLIC_KEY=FLWPUBK-...
```

## Deploy
- Recommended: Vercel or Netlify.
- Set `VITE_API_URL` in deployment settings to your Render backend.
